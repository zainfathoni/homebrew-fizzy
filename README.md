# Homebrew Tap for Fizzy Tools

Official Homebrew tap for [fizzy-md](https://github.com/zainfathoni/fizzy-md) and related Fizzy CLI tools.

## Installation

```bash
brew tap zainfathoni/fizzy
brew install fizzy-md
```
